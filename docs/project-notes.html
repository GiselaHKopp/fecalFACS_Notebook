30.05.2022 - information and recommendations from Gwen
Gwen

  • Gwen provides protocols:
    • recipe for DAPI stain RECIPE 005 - DAPI staining buffer.docx, We use this product: https://www.thermofisher.com/order/catalog/product/D1306?SKULINK #D1306, This is the manual: https://assets.thermofisher.com/TFS-Assets/LSG/manuals/mp01306.pdf
    • cell sorting protocol SOP #II.09 PME Protocol for Flow cytomety-JOE_rev06_GD.docx, We are also using a BD FACSAria so hopefully it will be easy to set it up with the same sorting parameters. The sample prep takes between 3-4 hours (depending on the sample) and I typically sort 2 to 3 samples at a time.
    • DNA extraction protocol SOP #II.14 PME Protocol Qiagen DNA mini kit with Small Volumes of Blood_GD.docx.
    • cytometer configuration settings. 130um nozzle 4 lasers configuration.csv, MELIN-GWEN-2C-ND-26OCT2021.pdf. We’ve been using neutral density filter 2.0 and a large 130um nozzle. The data I shared are from a run where we got 2593 cells for sample 707 and 454 cells for sample 712.
  • Gwen suggests to use a droplet digital PCR (ddPCR) for quantification of host DNA. We can also use qPCR but ddPCR is a bit more sensitive and can deal better with inhibitor prone samples. I am sharing the mammalian MYCBP primers we’ve been using for quantifying the host DNA below (doi: 10.1038/s41598-018-20427-9).
    • (update Gisela: we can use a ddPCR system in Lutz Becks’ lab, contact Ana del Caro ana.del-arco@uni-konstanz.de; details in del Arco et al. 2022)
    • Forward primer cmycF GCCAGAGGAGGAACGAGCT
    • Reverse primer cmycR GGGCCTTTTCATTGTTTTCCA
    • #1864008 DG8 Cartridges, QX100/QX200, 24/PK
    • #1863009 DG8 Gaskets, QX100/QX200, 24/PK
    • #12001925 ddPCR Plates 96-Well,Semi-Skirted,25/BX
    • #1814040 PCR Plate Heat Seal, foil, pierceable
    • #1864006 QX200 DG Oil for EvaGreen, 10 x 7 ml, 70 ml (10 x 7 ml), oil for use with the droplet generator in the QX200 Droplet Digital PCR System
